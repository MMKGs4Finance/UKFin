This repository presents the research outputs of the research project "Multimodal Knowledge Graphs as Infrastructures for Digital Twins of Financial Operational Risks: Mortgages and FX stocks".

(i)The ontology documentation can be found in the folder "version 4" in this repository. The public documentation, generated with WIDOCO is available at https://mmkgs4finance.github.io/UKFin/version4/index-en.html

(ii) Data and processing details can be found in the folder "Data". 

(iii)The resulting Multimodal Knowledge Graph is in the folder "MMKG".
